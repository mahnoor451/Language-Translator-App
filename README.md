# 🌐 Language Translator

A simple web app that translates text between multiple languages including English, Urdu, Sindhi, Pashto, and Punjabi.

## 🚀 Features
- Translate between 15+ languages
- Support for South Asian languages (Urdu, Sindhi, Pashto, Punjabi, Balochi)
- Clean, responsive interface
- Real-time translation

## 🛠️ Built With
- HTML5
- CSS3
- JavaScript
- MyMemory Translation API

2. Enter target language (e.g., "Urdu")
3. Type text to translate
